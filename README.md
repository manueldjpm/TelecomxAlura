# Desafío Telecom X  

## Objetivo del análisis  

El propósito de este desafío es apoyar a la empresa ficticia "Telecom X" en la identificación y comprensión de los factores que influyen en su elevada tasa de cancelación y pérdida de clientes.

## Organización del proyecto  

El proyecto se encuentra estructurado en los siguientes elementos:

- `Desafio_Telecom_X.ipynb`: Notebook principal donde se desarrolla el análisis, elaborado en Google Colab.  
- `README.md`: Documento que contiene la descripción y documentación general del proyecto.  

### Análisis contemplado  

El estudio se centrará en los siguientes puntos, en coherencia con los lineamientos del desafío:

- Análisis exploratorio de datos, tanto numéricos como categóricos.  
- Análisis de evasión o cancelación de clientes.  

> Los hallazgos (insights) y las visualizaciones se encuentran disponibles dentro del notebook.

## Instrucciones para ejecutar el notebook  

El análisis se desarrolla en un notebook de Google Colab, accesible mediante el siguiente enlace:

[Notebook de Colab: Desafío Telecom X](https://colab.research.google.com/drive/1UQ_IKGTIoRR7OUVdCLIPtlyyT2H5dQI4?usp=sharing)

Para ejecutarlo correctamente:

1. Accede al enlace para abrir el notebook en Google Colab.  
2. Revisa y sigue las indicaciones incluidas dentro del propio notebook.  
3. No es necesario descargar los datos manualmente, ya que el notebook realiza la carga automáticamente.  
4. Ejecuta cada celda para reproducir el análisis y obtener los resultados.
